## Dependencies
- Ruby v2.0+
- Ruby gem

## Install
```
gem install bundler
bundle install
```

## Start
```
bundle exec rails s -p8888 -b0.0.0.0
# -b0.0.0.0 Is to allow access from LAN or else only localhost can access
```

The server listens on port 8888: (http://localhost:8888)[http://localhot:8888]
